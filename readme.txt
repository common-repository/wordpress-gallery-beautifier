== Description ==

The plugin does make a better default wordpress gallery.



== Installation ==

* 1) Copy folder with this plugin to wp-content/plugins directory
* 2) Activate plugin
* 3) Now, your existing galleries, that’s displayed with the [gallery] tag becomes better :)

== Frequently Asked Questions ==

[English] How to set custom CSS for the plugin?
Just place a file with name wpcoder-gallery-beautifier.css in your theme directory, and set custom CSS rules for the gallery.
                
[Russian] Как изменить CSS стили для галереи?
Просто создайте файл с именем wpcoder-gallery-beautifier.css в директории вашей активной темы, и в нем пропишите новые стили для галереи.
